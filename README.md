# End-to-End Retail Analytics

## Overview
This project demonstrates a complete retail analytics pipeline using the **Online Retail dataset**.  
It covers:
- Data cleaning and feature engineering
- KPI calculation (Revenue, AOV, CLV)
- Customer segmentation using RFM and KMeans clustering
- Revenue leakage analysis
- Monthly revenue forecasting with rolling forecasts
- Executive dashboard visualization

## Dataset
- Public dataset from [Kaggle: Online Retail](https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset)

## Tools & Libraries
- Python, Pandas, NumPy, Matplotlib, Seaborn
- Scikit-learn for clustering
- Google Colab for execution

## Outcome
- Actionable insights for improving revenue and customer retention
- Resume-ready end-to-end analytics project
